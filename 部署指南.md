## 关于中文版

WhatCable 中文版已完全汉化，包括：

- ✅ **UI 界面**：所有菜单、按钮、提示均已翻译为中文
- ✅ **文档**：README、贡献指南等均已汉化
- ✅ **本地化文件**：`.xcstrings` 文件包含完整中文翻译

### 系统要求

- macOS 14.0 或更高版本
- 支持 USB-C / 雷电端口的 Mac

## 方法一：直接下载（推荐）

### 步骤

1. 访问 [GitHub Releases](https://github.com/Cao150702/whatcable/releases/latest)
2. 下载最新的 `WhatCable.zip`
3. 解压缩
4. 将 `WhatCable.app` 拖到 `/Applications` 文件夹
5. 首次运行时，右键点击应用 → "打开"（绕过 Gatekeeper）

### 注意事项

- 确保系统语言设置为**中文（简体）**
- 应用会自动以中文显示界面

## 方法二：从源码构建

### 前提条件

- Xcode 15.0 或更高版本
- Swift 5.9 或更高版本
- [XcodeGen](https://github.com/yonaskolb/XcodeGen)（可选，用于生成 Xcode 项目）

### 构建步骤

#### 1. 克隆仓库

```bash
git clone https://github.com/Cao150702/whatcable.git
cd whatcable
```

#### 2. 打开项目

```bash
# 方法 A：使用 XcodeGen 生成项目文件
xcodegen generate
open WhatCable.xcodeproj

# 方法 B：直接打开 Package.swift
open Package.swift
```

#### 3. 设置本地化（重要！）

确保 Xcode 项目正确识别中文本地化：

1. 在 Xcode 中，选择 **Project → Info → Localizations**
2. 确认 **Chinese (Simplified)** 已添加
3. 确认 `Localizable.xcstrings` 文件在本地化资源中

#### 4. 构建并运行

```bash
# 使用 Xcode
# 1. 选择目标设备（My Mac）
# 2. 点击 ▶️ 按钮运行

# 或使用命令行
xcodebuild -scheme WhatCable -destination 'platform=macOS' build
```

#### 5. 导出应用

1. 在 Xcode 中，选择 **Product → Archive**
2. 归档完成后，点击 **Distribute App**
3. 选择 **Copy App** → 导出到本地
4. 将 `WhatCable.app` 移动到 `/Applications`

## 方法三：Homebrew 安装

### 安装步骤

```bash
# 添加 tap（如果尚未添加）
brew tap Cao150702/whatcable

# 安装中文版
brew install --cask whatcable-zh
```

### 注意事项

- Homebrew 版本可能需要等待维护者更新
- 建议优先使用**方法一**（直接下载）

## 确保中文显示

### 检查系统语言设置

1. 打开 **系统设置 → 语言与地区**
2. 确保 **首选语言** 列表顶部是 **中文（简体）**
3. 如果没有，点击 **+** 添加，并拖到顶部
4. 重启应用（无需重启系统）

### 验证本地化

运行应用后，检查：

- ✅ 菜单栏图标右键菜单是否为中文
- ✅ 设置窗口是否显示中文
- ✅ 通知消息是否为中文

如果仍然显示英文，尝试：

```bash
# 强制设置应用语言（临时测试）
defaults write uk.whatcable.whatcable AppleLanguages '(zh-Hans)'
```

然后重启应用。

## 常见问题

### Q1: 应用显示英文而不是中文？

**A**: 确保系统首选语言是中文（简体）。如果问题依旧，尝试：

1. 删除应用并重新下载
2. 运行 `defaults delete uk.whatcable.whatcable` 重置偏好设置
3. 重新启动应用

### Q2: 构建时提示"找不到本地化文件"？

**A**: 确保 `.xcstrings` 文件已正确添加到 Xcode 项目中：

1. 在 Xcode 中，选择 `Localizable.xcstrings`
2. 在 **File Inspector** 中，确保目标成员已勾选（√ WhatCable, √ WhatCableCore）

### Q3: 如何贡献翻译？

**A**: 欢迎提交 PR！

1. 编辑 `.xcstrings` 文件
2. 在 `localizations` 下添加或修改 `zh-Hans` 条目
3. 提交 PR 到 https://github.com/Cao150702/whatcable

### Q4: 中文版会跟随官方版更新吗？

**A**: 是的！中文版是基于官方版 fork 的，会定期同步更新。

```bash
# 同步上游更新
git remote add upstream https://github.com/darrylmorley/whatcable.git
git fetch upstream
git merge upstream/main
```

### Q5: 如何分发给我朋友？

**A**: 有几种方式：

1. **分享 Release 链接**：发送 https://github.com/Cao150702/whatcable/releases/latest
2. **打包 DMG**：使用 `create-dmg` 工具创建安装包
3. **Homebrew**：如果你的朋友使用 Homebrew，可以让他们添加你的 tap

## 高级：创建发布包

如果你想自己创建发布包，按照以下步骤：

### 1. 构建 Release 版本

```bash
xcodebuild -scheme WhatCable \
  -configuration Release \
  -destination 'platform=macOS' \
  -archivePath ./build/WhatCable.xcarchive \
  archive
```

### 2. 导出应用

```bash
xcodebuild -exportArchive \
  -archivePath ./build/WhatCable.xcarchive \
  -exportPath ./build/export \
  -exportOptionsPlist exportOptions.plist
```

（需要创建 `exportOptions.plist` 配置文件）

### 3. 打包成 ZIP

```bash
cd ./build/export
zip -r WhatCable.zip WhatCable.app
```

### 4. 创建 GitHub Release

```bash
gh release create v0.6.0 WhatCable.zip \
  --title "WhatCable v0.6.0 中文版" \
  --notes "WhatCable 中文汉化版 v0.6.0"
```
